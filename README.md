# Random-Forest-Classifier
 Breast Cancer Prediction using Random Forest

## 📖 Overview
This project builds a Random Forest model to classify breast cancer tumors as malignant or benign.

## ⚙️ Model
- Random Forest Classifier
- OOB Score used for validation

## 📊 Results
- Train Accuracy: ~0.97
- Test Accuracy: ~0.96
- AUC Score: ~0.99

## 🔍 Key Insights
- Model shows low bias and controlled variance
- OOB score closely matches test accuracy
- shown feature importance
